Potions recipes
===============

Game for Ludum Dare 38

This game is about exploring the small world and creating potions.

Creating potions is a wonder.

Find all recipes!

Libraries used:
* PixiJS
* TweenMax
* SoundJS
* RequireJS
* spritesheet-js

Tools used:
* Paintbrush mac
* Tiled
* WebStorm
* GarageBand
* Audacity
